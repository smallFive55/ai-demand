# Story 1.3: 管理员配置角色与权限矩阵

## Story

**As a** 管理员,
**I want** 定义角色并配置菜单和操作级权限,
**So that** 不同角色只能访问与操作其授权范围内的数据和功能。

## Status: done

## Acceptance Criteria

- [x] **Given** 管理员已进入角色权限配置页 **When** 管理员创建或编辑角色并分配菜单与操作级权限 **Then** 权限规则立即生效到对应角色账号
- [x] **And** 系统支持按项目或业务线进行细粒度权限配置与校验
