3. 功能详细说明

3.1 转盘抽奖主页面 (UI)

3.2 抽奖逻辑机制 (后端交互)

条件分支 系统处理 前端反馈

积分不足 阻断交易 Toast：“您的积分余额不足”，引导去赚积分
